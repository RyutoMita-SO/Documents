## HTTP リクエストメソッド
参考：https://www.hypertextcandy.com/vue-laravel-tutorial-application-design  
参考：https://developer.mozilla.org/ja/docs/Web/HTTP/Methods  
WebAPIで使えるメソッドは下記５つ
|  メソッド  |      意味       |
|:-------:|:--------------:|
|  GET    | リソースの取得     |
|  POST   | リソースの新規作成  |
|  PATCH  | リソースの一部変更  |
|  PUT    | リソースの置き換え  |
|  DELETE | リソースの削除     |
